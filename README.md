# Archanova Solidity
[![NPM version][npm-image]][npm-url]

```bash
$ npm i @archanova/solidity -S
```

Environment variables:

| Name 	| Default Value 	|

| KOVAN_ACCOUNT_MNEMONIC 	| `-` 	|
